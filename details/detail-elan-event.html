Positive discipline Murcia Event 2020

This is a website for a Positive Discipline event in Murcia, Spain in 2020.
It has the following pages:

  • A main page with the program and the speakers.
  • A schedule page with the detailed schedule and the time table.
  • A about page with a small description of the event, the event that took place in 2019 and the partners.
  • A ticket page for purchasing items.
